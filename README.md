# Medical Document Intelligence Assistant

A domain-adapted AI system for extracting, summarizing and explaining information from medical documents.

## Initial capabilities

- Medical PDF text extraction
- Structured laboratory-result extraction
- Plain-language document summaries
- Evidence-based document question answering
- Page-level citations
- Medical safety restrictions

## Important limitation

This project is for educational and research purposes. It does not provide medical diagnosis, prescribe treatment or replace a qualified healthcare professional.

## Technology

- Python
- FastAPI
- MedGemma 1.5 4B
- Hugging Face Transformers
- ChromaDB
- Sentence Transformers
- React and Vite
- QLoRA fine-tuning
